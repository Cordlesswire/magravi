Valutazione del traffico per HTTP, FTP/SFTP, e-mail e WebDAV

La valutazione illustrata di seguito mostra il trasferimento dei dati nei singoli mesi dell’anno corrente. I dati sono suddivisi tra HTTP, FTP, e-mail e WebDAV. La somma dei valori singoli viene indicata nelle prime due colonne. Il numero degli accessi indica quante risposte HTTP o caricamenti/download da FTP derivanti dal proprio pacchetto (per tutti i domini) sono stati richiesti.

Requisito fondamentale per le risposte HTTP è il file access.log.., mentre per gli accessi FTP/SFTP sono i file ftp.xferlog e sftp.log. Tali file si trovano in Directory /logs (ftp.xferlog solo per i pacchetti con particolari IP). 'Numero' nella sezione della posta elettronica indica quante e-mail sono state inviate in determinati mesi con gli script della propria presenza. Le e-mail inviate tramite mailing list non verranno conteggiate.

I megabyte, che vengono associati ai singoli punti della struttura, indicano quanti megabyte sono stati effettivamente utilizzati attraverso il numero elencato per gli accessi o le e-mail.
